# Balls and Bins: A Load Balancing Simulation

This repository contains a Jupyter Notebook for an empirical study of "balls and bins" allocation strategies. The notebook simulates and analyzes the performance gap of various allocation schemes, including 1-choice, 2-choice, (1+$\beta$)-choice, batched, and partial information strategies.
